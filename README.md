## Installation

Using Yarn:

```bash
$ yarn global add node-green-cli
```

Using npm:

```bash
$ npm install -g node-green-cli
```

## Usage

In your terminal:

```bash
$ node-green Array.prototype.includes
$ node-green --node-version=6.4.0 Array.prototype.includes
$ node-green --allow-harmony Array.prototype.includes
```
